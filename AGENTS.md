# Agent notes

## Tooling
- Use `uv` for Python environment management and installs (for example,
  `uv venv`, `uv pip install`, and `uv run`).

## ADK testbed
- The OpenCode testbed expects `OPENCODE_ZEN_API_KEY` plus
  `OPENAI_API_KEY` and `OPENAI_BASE_URL` (set to
  `https://opencode.ai/zen/v1`) before running evals.
